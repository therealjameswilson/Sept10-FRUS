Targets searched:

- <https://www.archives.gov/declassification/iscap>

- <https://www.archives.gov/files/declassification/iscap/appeals-log-q1-2026.xlsx>

The ISCAP main page is mostly a routing page: it explains ISCAP's Executive
Order 13526 function, links to the quarterly release page, links to the
decisions page, and posts the current appeals log. The release table remains the
real public document surface. The decisions page routes users back to the
release table rather than exposing a separate document list. The Q1 2026 appeals
log has several undetailed FBI appeal rows, but it does not identify released
9/11/counterterrorism document titles beyond the release table.

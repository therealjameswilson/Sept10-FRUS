# Retrieval Matrix

Prepared May 23, 2026.

This matrix converts the compiler gap assessment into record-level controls. It does not mean every archival gap has been solved. It means every current record now has a retrieval priority, compiler-risk level, scope role, retrieval status, PDF role, and next action.

## Counts

| Priority | Count | Meaning |
| --- | ---: | --- |
| P0 | 10 | Blocking before treating the lead as a selected FRUS document. |
| P1 | 13 | High-value retrieval or verification needed for balance and annotation. |
| P2 | 3 | Supporting source shelf or endpoint context. |
| Resolved | 11 | Direct public copy or strong public text located; annotation still required. |
| Reference | 12 | Recollection, report, or background source; not a selection driver. |
